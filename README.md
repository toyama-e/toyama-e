## 開発中・制作実績

### 🏯 [いさんぽJAPAN](https://github.com/toyama-e/japan-heritage-guide.git)
「知識がなくても日本の世界遺産を楽しめる」をコンセプトに、歴史を知るだけでなく、実際に足を運び、記録する体験をシームレスにつなぐWebアプリです。

こだわりポイント:
- ゲーミフィケーション: 訪問記録に応じたバッジ獲得や称号アップ機能を実装。バッジ獲得時の演出など、ユーザーが達成感を得られる体験を設計しました。
- フロントエンド: APIからの訪問状況をリアルタイムに反映し、バッジをモノクロからカラーへ動的に変化させるロジックを実装しました。
- 技術スタック: Next.js / FastAPI / PostgreSQL / Firebase Auth

[GitHubリポジトリを見る](https://github.com/toyama-e/japan-heritage-guide.git)
