## プロフィール

15年間の看護師経験を経て、Webエンジニアへキャリアチェンジしました。
急性期などのミスが許されないスピード感のある現場で、多職種と連携し、認識がズレないよう調整する役割を担ってきました。 この経験で培った「徹底的なユーザー理解」と「状況に応じた完遂力」を武器に、ユーザーに価値を届け続けるプロダクト開発を目指しています。

## 技術スタック

- Frontend：Next.js / TypeScript

- Database：PostgreSQL（Supabase）
- Auth：Firebase Authentication

- 利用者や関係者の背景を理解したうえで、要件や仕様を整理すること
